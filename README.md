Asignacion Proyecto(Extra)

#Caracteristicas:
El siguiente proyecto consiste en el desarrollo de un sistema de reservas de asientos en dos autobuses utilizando el lenguaje 
de programacion Java y el uso de hilos (threads). Nuestro objetivo principal es simular la compra de boletos de manera concurrente, 
validando asi la disponibilidad de asientos, controlando errores de entrada y mostrando una carrera final entre los autobuses
dependiendo de la cantidad de pasajeros registrados.  
El sistema permite al usuario elegir el autobus, seleccionar el asiento, validar que este disponible y decidir si desea seguir 
registrando compras y finalmente mostramos una animacion de carrera donde el bus con menos pasajeros(el ms ligero) tiene mayor probabilidad de ganar.

#Funciones:
Funcion 1: Simular cual es la compra de boletos de manera concurrente utilizando los hilos.  
Funcion 2: Validar cuales son las entradas del usuario para evitar asi los errores en la seleccion de autobus y de asiento.  
Funcion 3: Controlar la disponibilidad de los asientos y evitar que haya una sobreventa.  
Funcion 4: Simular una carrera final basada en la cantidad de personas registradas en cada uno de los autobuses.

#Ejercicios:
- Registro manual de boletos por medio de teclado.
- Seleccion de autobus (MONSTER o RED BULL).
- Seleccion de asiento del 1 al 10 con validacion.
- Uso de hilos para simular clientes comprando boletos.
-  Simulacion de carrera de autobuses con dibujo en la consola.
